---
title: "New Website Launched for GitHub Pages"
date: 2026-06-10
---

The website of the Forecast, Analysis, Application Laboratory has been redesigned and refreshed, with Hugo, which is a static site generator written in Go, as its backbone. The website is now hosted on GitHub Pages.

The new website includes a “Forecasts” section, where experimental aviation meteorology forecasts will be made available in the near future.
